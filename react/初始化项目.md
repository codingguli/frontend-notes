# 初始化项目

## 一、通过脚手架创建项目


```bash
  yarn create vite
```

## 二、配置路径别名

1. **添加@types/node依赖**

```bash
pnpm add @types/node -D
```

2. **在vite.config.ts新增配置（路径别名）**
```ts
export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, 'src'),
    }
  },
})
```
3. **在tsconfig.json/tsconfig.app.json中新增配置（路径提示）**

```json
{
  "baseUrl": ".",
  "paths": {
    "@/*": ["src/*"]
  }
}
```
